# VisionPool_Capstone2026-2027

VisionPool is a proposed real-time pool rule compliance monitoring system that uses object detection to 
automatically identify violations such as missing wristbands, improper swimming attire, and prohibited jumping 
or diving activities. The system captures screenshots of flagged incidents and transmits them to a web-based 
administrative dashboard, providing pool staff with an efficient, data-driven tool for rule enforcement and 
incident documentation.
